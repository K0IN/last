# last

The last package on pypi (no really go take a look [pypi index](https://pypi.org/simple/) scroll to the bottom)

## what does it provide

- last lowercase letter: 'z'
- last uppercase letter: 'Z'
- last digit: '9'
- last digit of pi: 3.1415926535....**0-9** (random will be right 1/10 of the time)
- last digit of e: 2.7182818284**0-9** (random will be right 1/10 of the time)
